# MenuEasy

**Short Description:**  
What is MenuEasy?
MenuEasy is a sleek and powerful in-game menu that lets you execute commands and useful actions instantly with a single click.
No typing, no hassle — just open the menu and control your world faster and smarter.
Designed for simplicity and speed, it brings everything you need into one clean, neon-styled interface.
